# Machine Learning & Deep Learning — Two-Week Intensive Course

**Instructor:** Kieran Didi  
**Dates:** 17–28 August 2026 (Mon–Fri × 2 weeks)  
**Format:** 3 hours lecture + practical per day

Each day has **lecture notes** (derivations), **PDF slides** (figures from MML, UCL/DeepMind, MIT 6.S184, and course materials), and an **exercise** (Jupyter notebook).
